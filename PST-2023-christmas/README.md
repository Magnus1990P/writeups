# NPST 2023
Julenissen kaller alle alveteknologer inn til taskforce alv!

Denne består av hjelpere fra Nordpolar sikkerhetstjeneste (NPST), Nissens sikkerhetsmyndighet (NISM) og Kriminalvpolitisentralen (Kriapos).
